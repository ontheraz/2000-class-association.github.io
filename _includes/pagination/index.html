{%- comment -%} This file serves as the root of the pagination workaround I've had to write. The bug was as logged here: https://github.com/sverrirs/jekyll-paginate-v2/issues/209 The work around is to have every file using pagination call this file which then sorts by the page title (and thus the collection to be displayed) and then loads the file for the right one. {%- endcomment -%} {% if page.title contains 'Events' %} {% include pagination/events.html eventsPaginator=include.pagIndexPaginator %} {% endif %} {% if page.title contains 'Gallery' %} {% include pagination/gallery.html galleryPaginator=include.pagIndexPaginator %} {% endif %} {% if page.title contains 'Results' %} {% include pagination/results.html resultsPaginator=include.pagIndexPaginator %} {% endif %}
